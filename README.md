# OTUS React.js Developer

Проект для курса OTUS React.js Developer.

## Описание

Этот репозиторий содержит учебный проект, разрабатываемый в рамках курса "React.js Developer" от образовательной платформы OTUS.

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/termen911/otus-react.git
cd otus-react

# Установка зависимостей
npm install
```

## Запуск

```bash
# Запуск основного приложения
npm start

# Проверка версии Node.js
npm run node:version
```

## Структура проекта

```
otus-react/
├── index.js           # Основной файл приложения
├── showNodeVerison.js # Скрипт для проверки версии Node.js
├── package.json       # Конфигурация проекта и зависимости
└── README.md          # Документация проекта
```

## Доступные скрипты

- `npm start` - Запуск основного приложения
- `npm run node:version` - Проверка версии Node.js
- `npm test` - Запуск тестов (пока не настроено)
